01 / Deployed · Pseudocode

Collaborative Filtering Engine

Uses TF-IDF term weighting combined with Word2Vec semantic similarity to generate personalised product recommendations.

# ── Collaborative Filtering with TF-IDF + W2V ── FUNCTION build_item_profiles(catalogue): FOR EACH item IN catalogue: tokens = tokenise(item.description) tfidf_vec = tfidf_transform(tokens) w2v_vec = mean_word_vectors(tokens, w2v_model) item.embedding = concat(tfidf_vec, w2v_vec) RETURN item_matrix FUNCTION score_candidates(user, candidates): user_profile = build_user_profile(user.interactions) scores = [] FOR EACH item IN candidates: sim = cosine_similarity(user_profile, item.embedding) cf_score = collaborative_score(user, item) # Weighted blend of content + CF signal final = 0.6 * sim + 0.4 * cf_score scores.append((item, final)) RETURN sort_descending(scores) FUNCTION recommend(user, top_k=10): candidates = filter_unseen(user, all_items) ranked = score_candidates(user, candidates) RETURN ranked[:top_k]

03 / Deep Learning · Pseudocode

Neural Collaborative Filtering

Two-tower neural architecture — one tower encodes users, one encodes items — and learns to maximise similarity between matched pairs.

# ── Two-Tower Neural CF Model ── MODEL TwoTowerRecSys: user_tower(user_id, user_features): x = Embedding(user_id, dim=128) x = concat(x, user_features) x = Dense(256, activation="relu")(x) x = Dense(128, activation="relu")(x) RETURN L2_normalise(x) # user embedding item_tower(item_id, item_features): x = Embedding(item_id, dim=128) x = concat(x, item_features) x = Dense(256, activation="relu")(x) x = Dense(128, activation="relu")(x) RETURN L2_normalise(x) # item embedding forward(user, item): u = user_tower(user) i = item_tower(item) score = dot_product(u, i) RETURN score # Training: maximise score(user, pos) - score(user, neg) train(triplets, loss="BPR", epochs=20)

04 / Evaluation · Pseudocode

Offline Evaluation Framework

A rigorous offline evaluation pipeline measuring recommendation quality across precision, recall, NDCG and catalogue coverage — without needing live traffic.

# ── Offline Evaluation Pipeline ── FUNCTION evaluate_recommender(model, test_set, k=10): metrics = {precision:[], recall:[], ndcg:[], coverage:set()} FOR EACH (user, ground_truth) IN test_set: recs = model.recommend(user, top_k=k) # Precision@K: relevant items in top-K hits = len(intersect(recs, ground_truth)) metrics.precision.append(hits / k) # Recall@K: how much of relevant set was found metrics.recall.append(hits / len(ground_truth)) # NDCG@K: rank-aware quality score metrics.ndcg.append(ndcg_score(recs, ground_truth, k)) # Coverage: unique items recommended metrics.coverage.update(recs) RETURN { "precision@K": mean(metrics.precision), "recall@K": mean(metrics.recall), "ndcg@K": mean(metrics.ndcg), "coverage": len(metrics.coverage) / len(all_items) }
NDCG@KPrecision@KRecall@KCoverage
